# tf-summarize Workspace Instructions

## Repository Context
This is a Go CLI tool (`tf-summarize`) that summarizes Terraform plan output.
Pipeline: reader → parser → writer. Each stage has a minimal interface.

## Code Style
- Go 1.26, standard library preferred, no CGO
- `fmt.Errorf` for error wrapping, never `errors.Wrap`
- File permissions: `0600`
- Interfaces: 1-2 methods max
- Test assertions: `testify/assert`
- Deterministic output ordering: use explicit order slices (e.g., `tableOrder`, `htmlChangeOrder`)

## Testing
- Co-located `*_test.go` in each package
- `bytes.Buffer` to capture writer output
- Mock `NewTreeWriterFunc` for tree isolation
- Integration tests compile the binary and run as subprocess
- Run: `go test -v ./... -count=1`

## Common Tasks
- Adding a new output format: implement `Writer` interface, add constructor, wire into `CreateWriter()` factory in `writer/writer.go`
- Adding a new input source: implement `Reader` interface, wire into `CreateReader()` in `reader/reader.go`
- Adding a new change type: update `terraformstate/terraform_state.go`, add to order slices in writer packages
